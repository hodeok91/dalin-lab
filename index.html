downloads

exe, apk, zip 파일은 GitHub Releases에서 버전별로 관리합니다.

다운로드 운영 원칙
웹에서 바로 실행되는 HTML/CSS/JS 웹앱은 apps 폴더에 올리고, Windows용 exe, Android용 apk, zip, PDF 설명서는 GitHub Releases에 올립니다. 이렇게 나누면 홈페이지는 가볍게 유지되고, 설치파일은 버전별로 안전하게 관리할 수 있습니다.

installGuide

설치와 실행 방법을 안내할 자리입니다.

TERMINAL · 설치방법 예시
$ open dalin-lab/releases # 1. 필요한 버전의 exe 또는 apk를 다운로드합니다. # 2. 학교 PC에서는 보안 정책 때문에 실행이 제한될 수 있습니다. # 3. Android apk는 알 수 없는 앱 설치 허용이 필요할 수 있습니다. # 4. 개인정보가 들어가는 앱은 사용 전 안내문을 확인합니다.
